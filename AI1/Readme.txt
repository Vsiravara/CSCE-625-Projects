To run the program from command line enter
>python AI.py <filename> <x coordinate of source> < y coordinate of source> < x coordinate of destination> < y coordinate of destination >

Eg. If you want to run the search algorithms from point (1,1) to (20 , 20 ) and the graph is stored in file named "ATM.graph"
python AI.py ATM.graph 1 1 20 20
Command prompt view :
